## Installation

**Clone repo**

```bash
git clone https://github.com/codejiva/panenIN.git
cd panenIN
```
*1. Backend*

**Install project with npm**

```bash
cd backend-agribud
npm install
```

**Configuration**
```javascript
// config.js
module.exports = {
  geminiApiKey: 'PASTE_API_KEY_MAKERSUITE_HERE'
};
```

**Start server**
```
npm run dev
```
if success:
```
Server jalan di port 5000 ya, warga!
```
